# HuggingFace Pipeline Use

- 주제 : 영화 시놉시스로 예측해보는 영화 장르
- 설명 :
  - 시놉시스는 영화의 주제와 줄거리, 분위기 등을 짧게 요약한 글로, 영화에 관심이 있는 관객이 실제로 해당 영화를 선택하는 데 있어 중요한 요소 중 하나입니다.
  - 10편의 영화를 선정하여 시놉시스를 통해 분류 모델이 장르를 분류한 결과를 확인하고 실제 장르와의 차이는 어느 정도인지 알아보았습니다.
    
- 대상 : 2026/01/01 ~ 2026/03/14 국내 박스오피스 상위 1 ~ 10위 영화
- Task : zero-shot-classification
- 선정 모델 : facebook/bart-large-mnli
- 장르 : 'SF', '가족', '공연', '공포', '다큐멘터리', '드라마', '멜로/로맨스', '뮤지컬', '미스터리', '범죄', '사극', '스릴러', '애니메이션', '액션', '어드벤처', '전쟁', '코미디', '판타지' = 총 18개
-----------------------------------------------------------------------------------------------------------------------------------
- 분류 결과
  - 분류 Score를 높은순으로 정렬한 것을 '분류 순위'라고 할 때, 다수 영화의 경우 실제 장르의 분류 순위가 높은 편임을 알 수 있습니다.
  - 총 18개 장르가 Score 총합 1을 나눠 갖는 구조 때문에 분류 순위가 높아도 Score는 그닥 차이가 나지 않습니다.
  - 'SF'라는 단어를 인식하지 못 하는지 'SF' 장르는 줄곧 최하위를 기록했습니다.
